# Login-Form
Created a resposive Login Form using HTML and CSS.

# Introduction
This project provides a basic login form created using HTML. It includes fields for entering an email or phone number, a password, and options for social media login (Facebook and Twitter).

# Features
- Clean and simple design.
- Social media login options (Facebook and Twitter).
- Remember me checkbox.
- Forgot password link.
- Signup link for new users.

# Installation
No installation is required for this project. Simply open the HTML file in a web browser.

# Usage
- Open the HTML file in a web browser.
- Enter your email or phone number and password.
- Optionally, use the social media login options.
- Click the "Login" button.
